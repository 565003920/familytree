# 族谱网站 MVP 实施计划

## 项目概述

**项目名称**：族谱网站
**开发方式**：前后端分离全栈
**技术栈**：Spring Boot 3 + Vue 3 + MySQL
**开发策略**：MVP优先，迭代开发

## MVP范围

### 第一阶段核心功能
1. 用户认证系统
2. 家族管理
3. 成员信息管理
4. 基础家族树展示
5. 简单相册功能

## 技术架构

### 后端
- 框架：Spring Boot 3.x
- 数据库：MySQL 8.0
- 缓存：Redis 7.0
- 认证：JWT + Spring Security

### 前端
- 框架：Vue 3 + TypeScript
- UI组件：Element Plus
- 状态管理：Pinia
- 构建工具：Vite 4


## 实施步骤

### 步骤1：项目初始化（1-2天）
- 创建后端Spring Boot项目
- 创建前端Vue3项目
- 配置数据库连接
- 搭建基础项目结构

### 步骤2：数据库设计（1天）
- 创建核心表：users, families, family_members, family_relations
- 创建索引和外键约束
- 初始化测试数据

### 步骤3：用户认证模块（3-4天）
- 后端：用户注册/登录API
- 后端：JWT token生成和验证
- 前端：登录注册页面
- 前端：路由守卫


### 步骤4：家族管理模块（3-4天）
- 后端：创建/查询家族API
- 后端：家族成员CRUD API
- 前端：家族列表页面
- 前端：成员管理页面

### 步骤5：家族树展示（4-5天）
- 后端：家族树数据查询API
- 后端：关系管理API
- 前端：使用D3.js实现树形可视化
- 前端：节点交互功能


### 步骤6：相册模块（2-3天）
- 后端：相册创建和照片上传API
- 后端：文件存储（本地或OSS）
- 前端：相册列表和照片展示
- 前端：照片上传组件

### 步骤7：测试和优化（2-3天）
- 单元测试
- 集成测试
- 性能优化
- Bug修复

## 总工期估算

**总计：18-24天（约3-4周）**
